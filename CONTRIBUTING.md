# Como contribuir

Obrigado por querer contribuir.

Este projeto é simples e deve continuar fácil de entender.

## Regras básicas

- Use HTML, CSS e JavaScript puro.
- Mantenha a estrutura MVC atual.
- Escreva nomes de classes e IDs claros.
- Evite dependências novas.
- Evite código complexo sem necessidade.
- Teste abrindo o `index.html` no navegador.

## Enviando mudanças

1. Crie uma alteração pequena.
2. Confira se a página abre sem erro.
3. Explique o que foi alterado.
